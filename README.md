# CSE Week Festival Website

A website for CSE Week hosted by PlumbBob and SESB at University of Minnesota Twin Cities.

## Features

- **Festival Information**: Displays details about CSE Week celebration including workshops, networking events, tech talks, and competitions
- **Event Schedule**: Complete schedule of all week-long events with times and locations
- **RSVP System**: Interactive form allowing attendees to register for events
- **Attendee List**: Real-time display of registered attendees and their selected events
- **Responsive Design**: Mobile-friendly layout that works on all devices

## How to Use

1. Open `index.html` in a web browser
2. Browse the festival information and event schedule
3. Fill out the RSVP form with your details:
   - Full Name (required)
   - Email Address (required)
   - Affiliation (required)
   - Select at least one event to attend (required)
   - Dietary restrictions (optional)
4. Click "Submit RSVP" to register
5. Your registration will appear in the "Current RSVPs" section

## Technical Details

- Pure HTML, CSS, and JavaScript (no external dependencies)
- Client-side storage using localStorage
- Form validation and user feedback
- Responsive grid layout

## Files

- `index.html` - Main website structure
- `styles.css` - Styling and responsive design
- `script.js` - RSVP functionality and form handling
